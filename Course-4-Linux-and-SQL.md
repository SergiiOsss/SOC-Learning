# 🐧 Course 4: Tools of the Trade - Linux and SQL

## 🎯 Learning Objectives
This course focuses on foundational computing skills essential for a Security Analyst:
- **Linux:** Communicating with the OS via Command-Line Interface (CLI).
- **Bash Shell:** Navigating the file system and managing user authentication.
- **SQL:** Querying databases to filter and analyze security-related data.

## 📂 Modules
1. **Introduction to Operating Systems:** GUI vs. CLI.
2. **Linux Architecture:** Understanding Kernels, Shells, and Distributions.
3. **Bash Commands:** File management and permissions.
4. **SQL Databases:** Data retrieval and table joins.
