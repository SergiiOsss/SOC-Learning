### OWASP Security Principles
- **Secure Defaults:** Applications should be secure out-of-the-box.
- **Fail Securely:** If a system fails, it should default to the most secure state (e.g., block all traffic).
- **Don’t Trust Services:** Always verify third-party data and systems.
- **No Security by Obscurity:** Security must rely on robust controls (encryption, passwords), not just hidden details.
