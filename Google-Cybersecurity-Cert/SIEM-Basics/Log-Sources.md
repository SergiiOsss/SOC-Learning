# Log Data & SIEM Fundamentals 🛡️📊

## 🪵 Common Log Sources
1. **Firewall Logs:** Incoming/Outgoing connection attempts.
2. **Network Logs:** Device connections and traffic between services.
3. **Server Logs:** Application events (logins, file access, errors).

## 🚀 SIEM (Security Information and Event Management)
SIEM tools (like Splunk, IBM QRadar, or Chronicle) help analysts by:
- **Centralizing** all logs in one location.
- **Automating** alerts for suspicious activity.
- **Correlating** events from different sources to find complex attacks.
