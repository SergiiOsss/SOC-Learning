
## 🚪 Essential Ports & Protocols (The Interview List)
| Protocol | Port | Security Status | Use Case |
| :--- | :--- | :--- | :--- |
| **SSH** | 22 (TCP) | ✅ Secure | Remote terminal access |
| **Telnet** | 23 (TCP) | ❌ Insecure | Old remote access (Avoid!) |
| **SMTP** | 587 (TCP) | ✅ Secure (TLS) | Sending emails |
| **POP3** | 995 (TCP) | ✅ Secure (SSL) | Downloading emails (Local) |
| **IMAP** | 993 (TCP) | ✅ Secure (SSL) | Syncing emails (Server-based) |
| **DHCP** | 67/68 (UDP) | ⚠️ Management | Assigning IP addresses |
| **ARP** | None | ⚠️ Layer 2 | IP to MAC resolution |
