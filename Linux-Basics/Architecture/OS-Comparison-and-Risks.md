# Operating Systems Analysis

## OS Comparison Matrix
- **Windows:** Closed-source, high enterprise adoption, primary malware target.
- **macOS:** Partially open-source (Unix-based), strong hardware-software integration.
- **Linux:** Fully open-source, essential for SOC tools and server security.
- **Android/iOS:** Mobile-focused, Android (Open/Linux-based) vs iOS (Closed).

## Legacy OS Risks
- **Definition:** Outdated systems no longer receiving security patches.
- **Security Impact:** Permanent vulnerability to new exploits (Zero-days).
- **Use Case:** Often kept for compatibility with specialized industrial or medical hardware.
