# 💻 Operating Systems: Security Comparison

## 📋 Overview of Common OS
- **Windows:** Closed-source. Primary target for most malware due to high enterprise market share.
- **macOS:** Partially open-source. Unix-based system with strong built-in security features.
- **Linux:** Open-source. Essential for cybersecurity professionals for server management and security testing.
- **Android/iOS:** Mobile operating systems. Android is open-source (Linux-based), while iOS is a closed/partially open ecosystem.

## 🚨 Legacy OS Risks
A **Legacy Operating System** is an outdated system that is no longer supported or patched. 
- **Risk:** High vulnerability to new threats because security updates are discontinued.
- **Reason for Use:** Compatibility with specialized legacy software or embedded equipment.

## 🔍 Vulnerability Tracking Resources
To stay updated on OS security issues, analysts monitor:
- **MSRC:** Microsoft Security Response Center.
- **Apple Security Updates:** Security releases for macOS and iOS.
- **Ubuntu CVE Report:** Known vulnerabilities for Ubuntu Linux.
- **Google Cloud Security Bulletin:** Vulnerabilities affecting Google services.
