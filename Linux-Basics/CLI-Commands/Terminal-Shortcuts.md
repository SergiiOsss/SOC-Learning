# ⌨️ Linux Terminal Navigation & Shortcuts

## 🚀 Efficiency Shortcuts
- **TAB:** Auto-complete commands, file names, or directories.
- **UP ARROW (↑):** Cycle through command history to reuse previous entries.
- **CTRL + C:** Immediately terminate a running process or clear the current line.
- **CTRL + L** or `clear`: Clear the terminal screen for a fresh start.

## 🖱️ Cursor Movement
- **CTRL + A:** Move cursor to the **beginning** of the line.
- **CTRL + E:** Move cursor to the **end** of the line.
- **Left/Right Arrows:** Move character by character through a command.

## 📋 Copy & Paste (Qwiklabs specific)
- **CTRL + C:** Copy text from the instructions/browser.
- **CTRL + V:** Paste text into the terminal.
*Note: Ensure you grant clipboard permissions in the browser when prompted.*

## 💡 Best Practices
- **Check Progress:** Always use the "Check my progress" button in labs to verify task completion.
- **Don't End Early:** Never click "End Lab" until you see the green checkmark on Coursera, as progress will be lost.
